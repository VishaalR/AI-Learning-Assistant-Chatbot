# 🤖 AI Learning Assistant Chatbot

An NLP-based AI Learning Assistant Chatbot built using Python, NLTK, TF-IDF Vectorization, and Cosine Similarity. The chatbot helps users learn about Artificial Intelligence, Machine Learning, Python, Data Science, Web Development, Career Guidance, and more through natural language conversations.

---

## 📌 Project Overview

Chatbots are one of the most common applications of Artificial Intelligence and Natural Language Processing (NLP). This project implements a rule-based chatbot that understands user queries, identifies their intent, and provides relevant responses.

The chatbot uses:

- Text Preprocessing
- Tokenization
- Stopword Removal
- Lemmatization
- TF-IDF Vectorization
- Cosine Similarity
- Intent Classification

The project also includes a modern web interface built using Flask, HTML, CSS, and JavaScript.

## 🧠 How It Works

### Step 1: User Input

```text
"What is Machine Learning?"
```

### Step 2: Preprocessing

- Convert to lowercase
- Tokenize text
- Remove stopwords
- Lemmatize words

### Step 3: TF-IDF Vectorization

Convert text into numerical vectors.

### Step 4: Cosine Similarity

Compare user query with stored patterns.

### Step 5: Intent Classification

Identify the most similar intent.

### Step 6: Response Generation

Return a predefined response from the matched intent.
